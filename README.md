# Nexus App

Nexus is a React + Vite application with a simplified folder structure.

## Project Structure

- `public/`
- `src/`
  - `assets/` (`hero.png`, `react.svg`, `vite.svg`)
  - `components/` (`navbar.jsx`, `protectedroute.jsx`, `skillssection.jsx`)
  - `context/`
  - `pages/`
  - `app.css`
  - `app.jsx`
  - `index.css`
  - `main.jsx`
- `.gitignore`
- `README.md`
- `eslint.config.js`
- `firebse.config.js`
- `index.html`
- `package-lock.json`
- `package.json`
- `vite.config.js`

## Run

```bash
npm install
npm run dev
```
